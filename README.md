# Databases Scripts

Repository with dataset conversion, harmonization, canonical gait processing, and plotting scripts used in the project.

## Included

- Python scripts under `src/`
- Helper scripts under `scripts/`
- Project configuration files needed to run the code

## Excluded

- `mh_toolbox/`
- local datasets under `data/`
- temporary outputs and local backups

## Main Areas

- dataset conversion to Eurobench-like layouts
- canonical gait profile construction
- gait event processing and segmentation
- plotting and population-level analysis

## Notes

This repository is intended to publish project scripts without bundling local datasets or the internal `mh_toolbox` code.
